## language

设置项目使用的编程语言。

```lua
language ("lang")
```

### 参数

lang是语言标识符。某些语言需要模块以提供全面支持。

| 语言 | 模块          |
| ---- | ------------- |
| C    | 内置;始终可用 |
| C++  | 内置;始终可用 |
| C#   | 内置;始终可用 |
| F#   | 内置;始终可用 |
| D    | 内置;始终可用 |

### 应用于

项目配置

### 可用性

C，C ++和C＃在Premake 4.0或更高版本中可用。其他为5.0或更高版本。

### 示例

将项目语言设置为C ++

```lua
language "C++"
```
