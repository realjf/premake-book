## optimize

优化功能指定在构建目标配置时使用的优化级别和类型。

```lua
optimize "value"
```

如果未为配置设置任何值，则将执行工具集的默认优化（通常不进行优化）

### 参数

值指定所需的优化级别：

| value | 描述                             |
| ----- | -------------------------------- |
| Off   | 不会执行任何优化。               |
| On    | 执行一组平衡的优化               |
| Debug | 通过一些调试器逐步支持进行优化。 |
| Size  | 优化以最小的文件大小             |
| Speed | 优化以获得最佳性能。             |
| Full  | 全面优化                         |

### 应用于

项目配置

### 可用性

premake 5.0 

### 示例

```lua
optimize "Speed"
```
